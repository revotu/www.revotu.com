```Python
import sys

reload(sys)
sys.setdefaultencoding('utf8')
```

## 使用`__future__`模块 ##
从`__future__`模块导入`unicode_literals`，使用Python3的字符串默认编码是Unicode的特性
```Python
from __future__ import unicode_literals #放在文件首行
```
