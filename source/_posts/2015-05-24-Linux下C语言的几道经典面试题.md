## 1、如果在Linux下使用GCC编译器执行下列程序，输出结果是什么？ ##
```C
#include <stdio.h>
int main() {
    int i = 43;
    printf("%d\n",printf("%d",printf("%d",i)));
    return 0;
}
```
<!-- more -->
答案如下：
```
4321
```
## 2、C语言程序不使用任何条件运算符，打印出十次"Hello"？ ##
答案如下：
```C
#include <stdio.h>
int main(int argc, char *argv[]){
    int i=0;
    while(10 - i++)
         printf("Hello");
    return 0;
}
```
或是：
```C
#include <stdio.h>
int main(int argc, char *argv[]){
    int i;
    for(i=0;10 - i;i++)
    	printf("Hello");
    return 0;
}
```
## 3、如果在Linux下使用GCC编译器执行下列程序，输出结果是什么？ ##
```C
#include<stdio.h>
int main(){
    int a=300;
    char *p=(char *)&a;
    printf("%d\n",*p);
    printf("%d",*++p);
    return 0;
}
```
答案如下：
```
44
1
```

## 7、如果在Linux下使用GCC编译器执行下列程序，输出结果是什么？ ##
```C
#include <stdio.h>
#include <string.h>
 
int main(void) {
    int a[2000];
    char *p = (char *)a;
    int i;
    for( i = 0; i < 2000; i++)
        a[i] = -i - 1;
    printf("%d\n", strlen(p));
    return 0;
}
```
答案如下：
```
1020
```
